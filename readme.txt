== Frequently Asked Questions ==
= Can I use it to my existing theme? =
Sure! It'll work with any theme where Elementor Page Builder plugin installed.

= Could I use this plugin without Elementor Page Builder? =
No. Without Elementor Page Builder, it does not work.

= Does it support Elementor Pro? =
Certainly, you could use the dynamic fields if you install the Elementor Pro.

= Is it a free plugin or it has any Pro version? =
Ofcourse, it's a free plugin, and we don't have any premium version. We don't show you any go to pro box on your dashboard.
